# Film Life

**フィルムで暮らしを記録する** — *Live your life like it's a film*

フィルムカメラ入門知識と、フィルムで生活を記録することをテーマにした個人サイトプロジェクト。

## コンセプト

「Film」という単語の二重性（写真フィルム / 映画）を活かし、「日常を映画のように記録する」という世界観を表現するサイト。

## サイト構成

| カテゴリ | 内容 |
|----------|------|
| First Roll | フィルム入門ガイド |
| Shooting | 撮影の技術と考え方 |
| Gear | 機材知識 |
| Everyday Scenes | 暮らしの記録エッセイ |
| Film Diary | 写真日記 |
